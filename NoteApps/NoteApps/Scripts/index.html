Ejercicio 2

Incluir una pagina que mostraremos y ocultaremos mediante un boton.


Apuntes: ng-include, ng-show

Dos maneras de incluir una partial view, directiva ng-include dentro de un div, o directamente como un elemento.

<div ng-include="'Views/LoginView.html'"> </div> <ng-include src="'Views/LoginView.html'"> </ng-include>

Para mostra/ocultar utilizar la directiva ng-show

<div ng-show="true"> </div>
